API Documentation

Integrate our heart disease prediction model into your own applications using our REST API.

Base URL

https://your-domain.com/api

All API requests should be made to this base URL. Replace your-domain.com with your actual domain.

API Endpoints

GET /health

Check if the API and model are functioning correctly.

Response

{
  "status": "ok",
  "model_loaded": true
}

GET /model-info

Get information about the loaded prediction model.

Response

{
  "model_name": "HeartDiseaseClassifier",
  "created_at": "2025-01-15T12:00:00Z",
  "features": ["age", "sex", "cp", "trestbps", "chol", ...],
  "target_labels": [0, 1]
}

GET /model-performance

Get performance metrics for the prediction model.

Response

{
  "accuracy": 0.86,
  "precision": 0.83,
  "recall": 0.85,
  "f1_score": 0.84,
  "auc_roc": 0.89,
  "training_size": 303
}

GET /sample-input

Get a sample input structure for the prediction endpoint.

Response

{
  "sample_input": {
    "age": 0,
    "sex": 0,
    "cp": 0,
    "trestbps": 0,
    "chol": 0,
    ...
  }
}

POST /predict

Get a heart disease prediction based on input parameters.

Request Body

{
  "age": 63,
  "sex": 1,
  "cp": 3,
  "trestbps": 145,
  "chol": 233,
  "fbs": 1,
  "restecg": 0,
  "thalach": 150,
  "exang": 0,
  "oldpeak": 2.3,
  "slope": 0,
  "ca": 0,
  "thal": 1
}

Response

{
  "prediction": 1,
  "label": "Positive",
  "probability": 0.89
}

Feature Descriptions

Feature Description Type Values
age Age in years Numerical Any positive integer
sex Gender Categorical 0 = female, 1 = male
cp Chest pain type Categorical 0-3 (0: typical angina, 1: atypical angina, 2: non-anginal pain, 3: asymptomatic)
trestbps Resting blood pressure (mm Hg) Numerical Any positive integer
chol Serum cholesterol (mg/dl) Numerical Any positive integer
fbs Fasting blood sugar > 120 mg/dl Categorical 0 = false, 1 = true
restecg Resting electrocardiographic results Categorical 0-2 (0: normal, 1: ST-T wave abnormality, 2: left ventricular hypertrophy)
thalach Maximum heart rate achieved Numerical Any positive integer
exang Exercise induced angina Categorical 0 = no, 1 = yes
oldpeak ST depression induced by exercise relative to rest Numerical Float value
slope Slope of the peak exercise ST segment Categorical 0-2 (0: upsloping, 1: flat, 2: downsloping)
ca Number of major vessels colored by fluoroscopy Numerical 0-3
thal Thalassemia Categorical 0-3 (0: normal, 1: fixed defect, 2: reversible defect, 3: unknown)

Code Samples

JavaScript/Fetch

fetch('/api/predict', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    age: 63,
    sex: 1,
    cp: 3,
    trestbps: 145,
    chol: 233,
    fbs: 1,
    restecg: 0,
    thalach: 150,
    exang: 0,
    oldpeak: 2.3,
    slope: 0,
    ca: 0,
    thal: 1
  }),
})
.then(response => response.json())
.then(result => {
  console.log('Prediction result:', result);
})
.catch(error => {
  console.error('Error making prediction:', error);
});

Python/Requests

import requests
import json

url = "https://your-domain.com/api/predict"

data = {
    "age": 63,
    "sex": 1,
    "cp": 3,
    "trestbps": 145,
    "chol": 233,
    "fbs": 1,
    "restecg": 0,
    "thalach": 150,
    "exang": 0,
    "oldpeak": 2.3,
    "slope": 0,
    "ca": 0,
    "thal": 1
}

response = requests.post(url, json=data)
result = response.json()

print("Prediction result:", result)
